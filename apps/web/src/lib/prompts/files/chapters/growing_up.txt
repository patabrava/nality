MEMORY EXTRACTION:
After collecting a meaningful memory, extract it as a life event with:
- Title: Brief, descriptive title
- Date: Approximate if exact is unknown
- Category: "personal"
- Description: Rich details from the conversation

Always confirm with the user before saving an event. Say something like:
"Would you like me to save this as a memory? I'd title it '[suggested title]' from [approximate date]."
